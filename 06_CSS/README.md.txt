CSS adalah kepanjangaan dari Cascading Style Sheets yang berguna untuk menyederhanakan proses pembuatan website dengan mengatur elemen yang tertulis di bahasa markup.
Fungsi Css adalah Mempercepat Proses Desain, Halaman menjadi lebih cepat dimuat, Style lebih beragam daripada HTML
ada 3 cara menambahkan file CSS yaitu External, Internal, dan Inline
CSS Selector adalah pola yang digunakan untuk memilih elemen yang ingin di styling.
CSS Grouping adalah fungsi untuk mengelompokan beberapa selector dalam satu deklarasi style.
CSS Font adalah fungsi untuk mengatur style pada font di CSS.
CSS Margin & Padding adalah fungsi untuk membuat ruang disektitar elemen.
CSS Background adalah fungsi untuk mengatur style background di CSS.
CSS Link Event adalah fungsi untuk mengatur style pada elemen link di CSS.
CSS Display adalah fungsi untuk mengatur tampilan pada elemen.
CSS Table adalah fungsi untuk membuat style pada elemen table HTML.
